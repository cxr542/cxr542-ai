# AI 학습·실험 허브

김윤형의 AI 도구 학습·실험 카드 대시보드 (adri-fluid-dark 톤).

- **사이트:** https://cxr542.github.io/cxr542-ai/
- **프로필:** https://cxr542.github.io/

## 로컬 미리보기

```bash
# Python 3
python -m http.server 8080
```

브라우저에서 `http://localhost:8080/` 열기.

## 구조

- `index.html` — 랜딩
- `ai/services.json` — 카드 데이터
- `js/ai-home.js` — 카드·필터 렌더
- `ai_home.html` — 루트 리다이렉트
