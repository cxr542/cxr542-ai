Vision Font

시력표 5단계 테스트로 글자 크기·줄간격·자간을 맞춘 뒤, URL로 연 웹 기사에 CSS를 주입해 읽는 Expo 앱입니다. Cloud Learning Chatbot과 별도 프로젝트이며, 노안·저시력 사용자용 읽기 실험(MVP)입니다.

화면

  • 시력 테스트 — 잘 보여요 / 흐려요 (40pt → 14pt)
  • 맞춤 설정 — 미리보기·±2pt 조정·저장
  • 기사 읽기 — WebView + 맞춤 폰트 CSS
