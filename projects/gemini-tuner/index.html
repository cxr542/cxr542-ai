GeminiTuner

Google Gemini 웹 채팅에서 토큰·예상 비용을 실시간으로 보는 Chrome 확장 (FinOps 사이드 패널). 데이터는 브라우저에만 저장됩니다.

기능

  • 누적 토큰·예상 비용 (USD / KRW)
  • 월 예산 대비 사용률·경고 알림
  • API 키 연동 시 AI Studio 지출 동기화
  • 패널 드래그로 위치 변경

링크

  • GitHub — 소스 코드
  • 개인정보 처리방침 (Chrome Web Store용)
  • gemini.google.com (설치 후 사용)
